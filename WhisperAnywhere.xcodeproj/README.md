# WhisperAnywhere.xcodeproj bootstrap

This environment does not have a full Xcode installation (`xcodebuild` unavailable), so the project
bundle is not generated here.

To open and run the app in Xcode locally:

1. Open `Package.swift` in Xcode.
2. Xcode will synthesize the project/workspace and build target automatically.

If you need a checked-in `.xcodeproj`, generate it from a machine with full Xcode tooling.
