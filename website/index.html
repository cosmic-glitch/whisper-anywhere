Features

Focused where it matters

Fn hold-to-talk

Press and hold Fn to start. Release to stop and transcribe. No extra shortcuts to remember.

Fast feedback loop

Start chime, live mini-HUD, transcribing state, and clipboard fallback message when no cursor is available.

Practical insertion

Transcript is inserted into the active text target. If unavailable, text is copied to clipboard so nothing is lost.

Philosophy

Small, clear, useful

One core interaction

Use one key, one gesture, and stay in your current app.

No surprise editing

Transcription text is inserted as-is. Your words stay your words.

Minimal setup overhead

Configure API key and permissions once, then dictate anywhere.

FAQ

Quick answers

What does Native Whisper require?

macOS 14+, microphone permission, accessibility permission, input monitoring permission, and an OpenAI API key.

How do I start dictation?

Place your cursor in any text field, hold Fn, speak, and release Fn.

What happens if there is no active text cursor?

Your transcript is copied to clipboard and the HUD shows: No cursor found. Copied to clipboard.

Where can I follow development?

All source code and updates are on GitHub.
