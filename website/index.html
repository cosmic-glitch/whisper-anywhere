Features

Everything you need, nothing extra

Works in your apps

Dictate in docs, chat apps, email, browsers, and most text fields on macOS.

One-key dictation

Press and hold Fn to record. Release Fn to transcribe and insert text.

Clipboard fallback

If no text field is active, your transcript is copied to clipboard so you can paste it anywhere.

Philosophy

Clear data boundaries

No stored recordings or transcripts

Native Whisper does not store your recordings or transcribed text locally or on any third-party server.

Your key, your transcription request

Your dictation data is sent only to the OpenAI Transcription API using the API key you provide.

FAQ

Quick answers

What do I need to run Native Whisper?

macOS 14+, microphone permission, accessibility permission, input monitoring permission, and an OpenAI API key.

How do I start dictation?

Place your cursor in a text field, hold Fn, speak, then release Fn.

What if there is no active text cursor?

Your transcript is copied to clipboard and the on-screen status says: No cursor found. Copied to clipboard.

Where can I track updates?

Source code and updates are on GitHub.
