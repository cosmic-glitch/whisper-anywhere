Features

Works in your apps

Dictate in docs, chat apps, email, browsers, and most text fields on macOS.

One-key dictation

Press and hold Fn to record. Release Fn to transcribe and insert text.

Clipboard fallback

If no text field is active, your transcript is copied to clipboard so you can paste it anywhere.

Philosophy

No stored recordings or transcripts

Whisper Anywhere does not persist your audio recordings or transcript text in app storage or backend storage.

OpenAI-only transcription path

Your dictation request is sent to the OpenAI Transcription API through Whisper Anywhere backend endpoints.

FAQ

What do I need to run Whisper Anywhere?

macOS 14+, microphone permission, accessibility permission, input monitoring permission, and Google sign-in.

How do I start dictation?

Place your cursor in a text field, hold Fn, speak, then release Fn.

Do I need to enter my own OpenAI API key?

No. Hosted mode uses Whisper Anywhere server-side keys. You only sign in with Google.

What if there is no active text cursor?

Your transcript is copied to clipboard and the on-screen status says: No cursor found. Copied to clipboard.

Where can I track updates?

Source code and updates are on GitHub.
