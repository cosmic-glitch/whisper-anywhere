Step 1 of 4

Download and install

1

Install the app

  1. Download the .dmg file using the button above.
  2. Open the downloaded file.
  3. Drag Whisper Anywhere into your Applications folder.
  4. Launch it from Applications. The app appears in your menu bar (top-right of your screen).
First launch: A configuration window opens automatically. Follow the next steps to complete setup.
Screenshot: DMG install window — drag app to Applications

Step 2 of 4

Enter your OpenAI API key

Whisper Anywhere uses your own OpenAI API key. Your audio goes directly from your Mac to OpenAI — nothing is stored or routed through third parties.

2

Paste your API key

  1. Go to platform.openai.com/api-keys and create a key (or copy an existing one).
  2. In the Whisper Anywhere configuration window, paste the key and save.
Alternative: You can also set the OPENAI_API_KEY environment variable instead.
Screenshot: Configuration window with the API key field

Step 3 of 4

Grant macOS permissions

Whisper Anywhere needs three permissions to record audio, detect the Fn key, and insert text. You only need to do this once.

3

Enable all three permissions

macOS will prompt you for Microphone access. For Accessibility and Input Monitoring, click the buttons in the configuration window to open System Settings, then toggle Whisper Anywhere on.

Troubleshooting

Common issues

Nothing happens when I press Fn

Check that Input Monitoring is enabled in System Settings → Privacy & Security → Input Monitoring. You may need to toggle it off and on, then relaunch the app.

Text doesn't appear in the active app

Make sure Accessibility permission is granted. Some apps with custom text fields may not support direct insertion — in that case, the transcript is copied to your clipboard instead.

I get an API error

Verify your OpenAI API key is valid and has available credits at platform.openai.com/usage.

I can't find the app after launching it

Whisper Anywhere is a menu bar app — it has no Dock icon. Look for the microphone icon in the top-right area of your screen. If your menu bar is crowded, it may be hidden behind the notch.

How do I update the app?

Download the latest DMG from this page or from GitHub Releases, and replace the app in your Applications folder.
